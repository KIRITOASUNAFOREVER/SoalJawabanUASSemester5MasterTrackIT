It is known that the following statements: A=-A+B*(B-C)–C/D+D

From Above Statement, Create:
a) Directed Acyclic Graph (DAG)
b) Three Address Code
c) Quadruples
d) Triples

:: Solution ::

a) Directed Acyclic Graph(DAG):- DAG is a useful data structure for implementing transformation of basic blocks of three address code.

rules for DAG- leaf are lebeled by unique identifier either variable or constants.

- interior nodes are labeled by an operator.

- node are also given a sequence of identifier or label.

in above question first create three address code

-A+B*(B-C)-C/D+D

T1= -A

T2= B-C

T3= B*T2

T4= C/D

T5= T3-T4

T6= T5+D

T7= T1+T6

b)Three Address Code:- in three address code atmost three address are used to preesent any statement.

genral form:- a=b operator c // where a,b,c are operants

in above question three code are

-A+B*(B-C)-C/D+D

T1= -A // uniary A

T2= B-C

T3= B*T2

T4= C/D

T5= T3-T4

T6= T5+D

T7= T1+T6

c) Quadruples:- quadruples is a technique of representation of three address code.

d) Triples:- it is also technique of representation of three address code. but in this result column is not used. it is affected the result during options.

example- in operants column T1 is replace by location (0), T2 is replace by (1),T3 is replace by (2),T4 is replace by (3),T5 is replace by (4),T6is replace by (5).
